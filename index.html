About The Tool & Options

What is this tool?

This generator helps you create commands for yt-dlp, a command-line tool for downloading video and audio from yt.

Download Options Explained

  • Download Type: Choose what you want to grab. "Audio Only" is for music/podcasts. "Video + Audio" downloads a standard video file. "Video Only" is for getting silent clips.
  • Audio Format: Select the file type for your audio. Original (Best Available) downloads the native audio stream without re-encoding, preserving the highest possible quality. M4A and MP3 are best for compatibility. Opus often provides great quality at a smaller size.
  • Normalization: This fixes inconsistent volume levels between different songs. It requires re-encoding, so it cannot be used with the "Original" audio format.
    • Dynamic (dynaudnorm): Best for noisy environments (like a car). It makes quiet parts louder and loud parts quieter, so you don't have to touch the volume knob. It changes the song's internal dynamics.
    • Loudness (loudnorm): Best for creating a consistent playlist for home listening. It adjusts each song's overall perceived loudness to a standard level while preserving the original dynamics within the song. The settings used here (I=-16:TP=-1.5:LRA=11) are optimized for modern music listening, matching standards used by services like Apple Music and Spotify.
      • I=-16 sets the target average loudness to -16 LUFS (Loudness Units Full Scale), a modern standard for music.
      • TP=-1.5 sets the maximum true peak to -1.5 dB, leaving headroom to prevent clipping after encoding.
      • LRA=11 allows for a healthy dynamic range, so the music doesn't sound overly compressed.
  • Video Format: Choose the container for your video file. MP4 is universally compatible. MKV is more flexible and can hold more types of data, like multiple audio tracks or subtitles.
  • Embed Metadata/Thumbnail: Turning these on will save information like the song title, artist, and album art directly into the file, which is essential for organizing a music library.
  • Keep Original File (-k): If enabled, `yt-dlp` will not delete the intermediate files (like the separate video and audio streams it downloads before merging). This is useful for debugging if something goes wrong.
  • Output Path: Lets you specify a folder where your files will be saved, for example C:\Users\YourName\Music.
